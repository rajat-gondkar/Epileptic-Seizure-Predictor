# Access Notes for Restricted Datasets

## EPGP via dbGaP (phs000742.v1.p1)

**URL**: https://www.ncbi.nlm.nih.gov/projects/gap/cgi-bin/study.cgi?study_id=phs000742.v1.p1

**Access Requirements**:
- Approved dbGaP access request
- NIH eRA Commons account required
- Institutional Data Access Committee approval

**Use in Project**: Validation only — provides paired EEG + genetic data for 750 patients. 
If unavailable, the CTGAN-generated synthetic paired dataset will be used as a substitute.

**Status**: Not acquired (requires institutional access request)
